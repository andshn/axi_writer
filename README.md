# AXI Writer

## Структура

- `hdl/axi_writer.sv` — IP-ядро.
- `tb/axi_writer_tb.sv` — тестбенч.
- `tb/axi_ram.v` — AXI4 RAM (Alex Forencich).
- `constraints/timing.xdc` — задание тактовой частоты 100 VUц в соответствии с заданием
- `scripts/create_project.tcl` — создание проекта Vivado

## Примечания
1. В тестбенче используется готовый модуль axi_ram. Проверка осуществляется с меньшим объемом памяти. При 256 МБ из задания проверяются только тайминги.
2. В тестбенче не проверяется backpressure.
3. Теоретически можно немного ускорить скорость записи если начинать писать следующие данные не дожидаясь bvalid. Однако в этом случае необходимо выдавать данные с разными id и следить чтобы они не пересекались.
